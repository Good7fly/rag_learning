## 使用方式

1. 创建并启用虚拟环境（Windows PowerShell）

```powershell
py -3.10 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m ensurepip --upgrade
python -m pip install -U pip
```

2. 启动 Jupyter（如需）

```powershell
python -m pip install jupyter
jupyter notebook
```
